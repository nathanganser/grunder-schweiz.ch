# Gruender Schweiz

Gruender Schweiz ist ein redaktionelles Review- und Ranking-Portal fuer Schweizer Freelancer-Software.

Im Fokus stehen:

- Rechnungssoftware
- Buchhaltungssoftware
- Geschaeftskonten und Banking
- Zeiterfassung und angrenzende Admin-Tools

Die Site setzt auf redaktionelle Bestenlisten, Einzelreviews, Ratgeber und eine sichtbare Methodik mit Affiliate-Disclosure.

## Aktueller Stand

- Home auf Review-Portal-Positionierung umgestellt
- Navigation auf `Kategorien`, `Bestenlisten`, `Reviews`, `Methodik`, `Ratgeber`
- Legacy-Routen wie `pricing`, `login`, `signup`, `docs` und `changelog` fuer v1 deaktiviert oder umgeleitet
- Seed-Content fuer Rankings, Reviews, Methodik, Legal-Minimum und Ratgeber angelegt
- README und Go-live-Todo auf den Portal-Stand gebracht

Diese Liste basiert auf Live-Suchergebnissen fuer Schweizer Suchmuster rund um Rechnungssoftware, Buchhaltung, QR-Rechnung, Geschaeftskonten und Zeiterfassung. Die Priorisierung folgt drei Regeln:

- erstens: Keywords mit klarer Kauf- oder Vergleichsintention zuerst

- drittens: Keywords nur dann priorisieren, wenn sie zum redaktionellen Setup aus `Bestenlisten`, `Reviews` oder `Ratgeber` passen

Wichtig: Das sind Ziel-Keywords, nicht 50 gleich starke Keywords. Die obersten 10 bis 15 Begriffe sind die wichtigsten Money-Keywords. Die restlichen Begriffe stuetzen Topical Authority, interne Verlinkung und Long-Tail-Traffic.

### Was diese Keyword-Liste strategisch leistet

- Sie deckt den kompletten Funnel ab: `Bestenlisten` fuer Kaufintention, `Reviews` fuer Marken-Intent, `Ratgeber` fuer Informations- und Compliance-Intent.
- Sie staerkt den Schweiz-Fit, weil Keywords wie `QR-Rechnung`, `MWST`, `Milchbueechli`, `Einzelfirma` und `Geschaeftskonto Schweiz` nicht sauber ueber generischen DACH-Content abfangbar sind.
- Sie passt zur bestehenden Architektur. Die meisten Begriffe lassen sich ohne Neuausrichtung auf vorhandene Seitentypen mappen.
- Sie schafft eine klare interne Verlinkungslogik zwischen Guides, Vergleichen und Reviews.

## Setup

Abhaengigkeiten installieren:


```bash
pnpm dev
```

Typecheck:

```bash
pnpm exec nuxt typecheck
```

Lint:

```bash
pnpm exec eslint .
```

Production Build:

```bash
pnpm build
```
